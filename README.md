# CloudNest Bicep Platform

CloudNest is a hands-on Azure Infrastructure as Code project built with Bicep.

## Project Goal

The goal of this project is to design and deploy a professional Azure cloud environment using modular Bicep templates.

## What This Project Will Cover

- Azure resource groups
- Virtual networks and subnets
- Network security groups
- Storage accounts
- App hosting foundations
- Monitoring and diagnostics
- Modular Bicep structure
- Azure CLI deployment workflow
- Git and GitHub documentation workflow
